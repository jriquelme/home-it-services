# Home IT services

This project showcases serverless solutions using different AWS services.

## Deploying

This a python CDK project, perform the usual spells:

```
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

And deploy with:

```
$ cdk deploy
```
